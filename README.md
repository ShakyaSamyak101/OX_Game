# OXゲーム（Unity）

## 概要
このプロジェクトは、大学2年次のゼミ活動の一環として、UnityとC#を用いて開発したAI対戦型の○×ゲームです。各プレイヤーは最大3つまでしか記号を配置できず、4つ目を配置すると最も古く配置した記号が消える独自ルールを実装しています。

## 機能
- 人対人モード
- 人対ルールベースAIモード
- メインメニュー
- スコア管理機能
- 開始できるリトライ機能

## 使用技術
- Unity
- C#
- Visual Studio

## 担当内容
- ゲームロジックの設計・実装
- ルールベースAIの実装
- UI（メインメニュー・ゲーム画面）の作成
- 人対人・人対AIモードの実装
- デバッグおよび動作検証

## このプロジェクトで学んだこと
- Unityを用いたゲーム開発
- C#によるプログラミング
- ルールベースAIの設計・実装
- デバッグと問題解決の進め方

## 今後の展望
現在は、ルールベースAIと学習型AIの違いをより深く理解することを目的として、学習型AIを実装した戦略ゲームの個人開発に取り組んでいます。

## 実行方法
1. Unityでプロジェクトを開きます。
2. `Assets/Scenes` フォルダを開きます。
3. `Main Menu` シーンと `Game` シーンを開きます
4. `Game` シーンを右クリックし、**Unload Scene** を選択します。
5. `Main Menu` シーンをアクティブシーンに設定します。
6. Playボタンを押して実行します。
